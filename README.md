# OOracle - おおだろ判定システム

「判定の重み」を感じさせる、演出重視の判定システムです。

## 特徴
- **演出重視のUI**: 判定前の暗転と3秒間の「溜め」が緊張感を高めます。
- **弾幕演出**: 判定結果に合わせて画面全体にメッセージが流れます。
- **レスポンシブ対応**: スマートフォンでも快適に動作します。

## 使い方
1. 表示されたボタンから判定したい項目を選択します。
2. 背景が暗転し「判定が完了しました」と表示されたら、「判定を表示」を押します。
3. 3秒間の待機後、大迫力の判定結果が表示されます。

## ローカル開発環境の構築
```bash
npm install
npm run dev
```
